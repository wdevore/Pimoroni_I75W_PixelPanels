# Pimoroni_I75W_PixelPanels
Various Python and C/C++ projects for Pimoroni's I75 LED panel

# Display

The display's origin is at the top left.

```
        (0,0)          |
                       |
                       |
                       |
                       |
        ------------------------------- +X
                       |
                       |
                       |
                       |
                       |               (width-1,height-1)
                       +Y
```
